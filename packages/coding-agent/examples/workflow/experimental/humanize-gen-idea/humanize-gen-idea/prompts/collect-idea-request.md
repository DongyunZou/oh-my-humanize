Provide the loose idea for Humanize gen-idea.

You can paste plain text, or use simple fields:

```text
body: add repo-grounded long-context compression	n: 6
output: .humanize/ideas/context-compression.md
slug: context-compression
```

Fields:
- `body` / `idea` / `prompt`: idea text.
- `n`: direction count, 2-6, default 6.
- `output`: Markdown output path, optional.
- `slug`: output filename seed, optional.

Plain text is accepted as the idea body.
