Provide the draft for Humanize gen-plan.

You can paste plain Markdown, or use simple fields:

```text
draft: add repo-grounded long-context compression
output: .humanize/plans/context-compression-plan.md
mode: discussion
```

Fields:
- `draft` / `body` / `prompt`: plan seed text.
- `output`: Markdown output path, default `plan.md`.
- `input`: source label/path, optional.
- `mode`: `discussion` or `direct`, default `discussion`.

Plain text is accepted as the draft.
